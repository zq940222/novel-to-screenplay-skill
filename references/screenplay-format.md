# Hollywood Standard Screenplay Format

## Page Layout
- Font: Courier 12pt (monospaced)
- Margins: Left 1.5", Right 1", Top/Bottom 1"
- ~1 page = ~1 minute of screen time
- 90-page script ≈ 90-minute film

## Element Formatting

### Scene Heading (Slug Line)
ALL CAPS. Format: `INT./EXT. LOCATION - TIME OF DAY`

```
INT. COFFEE SHOP - DAY

EXT. CITY STREET - NIGHT

INT./EXT. CAR (MOVING) - CONTINUOUS
```

Time options: DAY, NIGHT, DAWN, DUSK, CONTINUOUS, LATER, MOMENTS LATER

### Action (Description)
Present tense. Describe only what the audience SEES and HEARS.

```
The door swings open. SARAH (30s, sharp eyes, worn leather jacket) steps inside, scanning the room with practiced efficiency.
```

- First appearance of a character: NAME in ALL CAPS + brief physical description + age
- No internal thoughts — only observable actions and sounds
- Keep paragraphs to 3-4 lines max

### Character Name
ALL CAPS, centered (3.7" from left margin).

```
                         SARAH
```

### Parenthetical
Below character name, in parentheses. Use sparingly.

```
                         SARAH
                    (whispering)
            I know what you did.
```

Common parentheticals: (beat), (into phone), (O.S.), (V.O.), (CONT'D)

### Dialogue
Below character name (or parenthetical), 2.5" from left, 2.5" width.

```
                         SARAH
            I didn't come here for the coffee.
```

### Transition
ALL CAPS, right-aligned. Use sparingly — modern scripts mostly use CUT TO: only when essential.

```
                                          CUT TO:
                                          FADE OUT.
                                          SMASH CUT TO:
```

### MONTAGE
```
MONTAGE - SARAH PREPARES FOR THE MISSION

-- Sarah studies blueprints at her kitchen table.
-- She assembles equipment in the garage.
-- She drives through the city at dawn.

END MONTAGE
```

### INTERCUT
```
INTERCUT - PHONE CONVERSATION

INT. SARAH'S APARTMENT - NIGHT
INT. DETECTIVE'S OFFICE - NIGHT

Sarah paces by the window. The Detective leans back in his chair.
```

## Special Notations

| Notation | Meaning | Usage |
|----------|---------|-------|
| (V.O.) | Voice Over | Narration not spoken in scene |
| (O.S.) | Off Screen | Character present but not visible |
| (O.C.) | Off Camera | Same as O.S. (alternate) |
| (CONT'D) | Continued | Same character speaks after action |
| SUPER: | Superimpose | On-screen text/titles |
| TITLE CARD: | Title Card | Full-screen text |
| FADE IN: | Fade In | Script opening |
| FADE OUT. | Fade Out | Script ending (period, not colon) |

## Script Structure (Three-Act)

| Act | Pages (90-min) | Content |
|-----|----------------|---------|
| Act 1 | 1-22 | Setup, inciting incident, first turning point |
| Act 2A | 23-45 | Rising action, midpoint |
| Act 2B | 46-67 | Complications, crisis, second turning point |
| Act 3 | 68-90 | Climax, resolution |

## Title Page Format
```
                        TITLE IN CAPS

                          Written by
                        Author Name


                                    Contact info
                                    on lower right
```
